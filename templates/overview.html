Brief summary

Leaffliction is a school project from school 42.
The goal of the project is to use multiple tools to analyse, augment, and classify a dataset comprised of plant leaf images with various diseases, using deep learning and computer vision techniques.

This web page aims to present the results of the project in a clean manner

The code for this whole project can be found on my github.
Feel free to take a look at the PDF file describing the project assignment (PDF, 3.4Mb)

Data analysis

The data for the project is a subset from Kaggle's PlantVillage dataset.
More accurately, it only uses the Apple and Grapes disease data. Let's look at some examples.

The dataset is very imbalanced, and a bit of augmentation and transformation was needed.

Deep learning

We settled on a CNN (Convolutional Neural Network) to solve this project smoothly.
The model was built and trained with Tensorflow's functions on the augmented dataset.
More precisely, the model's overall layer structure is as follows : [ Conv2D | MaxPooling2D | ReLU | ReLU | Softmax ].

The model was then saved and stored inside a .joblib file to be used both in further prediction/evaluation scenarios and on this website.
After training, the model manages a 90% accuracy rate on validation sets!
